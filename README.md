# GitHub Pages Homepage

このフォルダは、GitHub Pagesでホームページを公開するためのスターターです。

## 編集する場所

- `site/index.html`: ページの文章と構成
- `site/styles.css`: 見た目
- `site/script.js`: 小さな動き
- `.github/workflows/pages.yml`: GitHub Pagesへ自動公開する設定

## 公開の流れ

1. GitHubで新しいリポジトリを作成します。
2. このフォルダをそのリポジトリへpushします。
3. GitHubのリポジトリ設定で、PagesのSourceを「GitHub Actions」にします。
4. `main` ブランチへpushすると、`site` フォルダの内容が公開されます。

## Codexへの頼み方

例:

- 「トップページを私の自己紹介サイトに変えて」
- 「作品カードを5つに増やして」
- 「スマホで見やすいように調整して」
- 「GitHub Pagesに公開する前に表示確認して」
