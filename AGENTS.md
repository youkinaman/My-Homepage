# Codex Settings

この作業フォルダでは、GitHub Pagesで公開するホームページを作成・編集します。

## 基本方針

- 公開対象は `site` フォルダです。
- まずは静的なHTML、CSS、JavaScriptだけで作ります。
- 公開設定は `.github/workflows/pages.yml` を使います。
- 大きな変更の前には、既存ファイルを確認してから編集します。
- 変更後は、可能な範囲でローカル表示とGitの状態を確認します。

## よく使う依頼

- 文言、プロフィール、作品一覧、連絡先の差し替え
- レイアウトや配色の調整
- 画像やリンクの追加
- GitHub Pages公開前の最終チェック
